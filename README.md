# **ScanCamAnalyze**

### **AI-Powered Facial Health Scanning Application**

**ScanCamAnalyze** adalah solusi teknologi kesehatan (Health-Tech) inovatif yang dirancang untuk menganalisis kondisi kesehatan pengguna melalui pemindaian wajah menggunakan kecerdasan buatan. Proyek ini merupakan hasil kolaborasi lintas fungsi (Cross-Functional) yang dikembangkan sebagai **Capstone Project** dalam program **Bangkit Academy 2024**.
